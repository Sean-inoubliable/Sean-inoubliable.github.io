Nginx主要应用 - 动静分离

2020-07-16
  • Nginx
  • Nginx 的 负载均衡 和 静态代理 结合在一起, 可以实现 动静分离, 这是实际应用中常见的一种场景
  • 动态资源, 如 jsp 由 Tomcat 或其他 Web 服务器完成
  • 静态资源, 如 img / css / js 等由 Nginx 服务器完成
  • 充分利用其各自优势, 从而达到更高效合理的架构
