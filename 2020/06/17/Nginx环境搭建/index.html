安装

安装前准备

Nginx的安装 需要Linux安装相关的几个库, 否则配置和编译会出现问题

  1. gcc 编译器
  2. openssl库
  3. pcre库
  4. zlib库

正式安装

  1. 解压下载下来的 Nginx 文件
  2. 切换至解压后的 Nginx 主目录
  3. 指定路径安装
  4. 编译
  5. 安装

    安装后的文件夹

  • conf 配置, nginx.conf 核心
  • html 服务器默认访问模版页
  • logs 日志
  • sbin nginx 启动程序
